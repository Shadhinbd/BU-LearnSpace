Technologies Used:
❖ Frontend: HTML, CSS
❖ Backend: PHP
❖ Database: MySQL
❖ File Storage: Local server-based file handling


The BU LearnSpace system provides an efficient solution for managing and
distributing study materials across Bangladesh University. With secure
authentication, easy access to academic resources, and streamlined teacher–student
communication, the system enhances the overall learning experience. Built using
PHP and MySQL, it ensures reliable performance and can be expanded with
additional features in the future.Many students often miss important PDF or notes
because classroom codes are not shared properly, or resources get lost in messaging
apps. This platform ensures that no student is left behind. All academic materials
remain organized, accessible, and permanently available in one place.
